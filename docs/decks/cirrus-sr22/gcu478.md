# SR22 GCU478

## Overview { #sr22-gcu478-preview }

This page focuses on the G1000 control workflow and rotary-driven navigation interaction.

- The push buttons cover `DIRECT`, `MENU`, `FPL`, `PROC`, `CLR`, `ENT`, `HOME`, and frequency transfer.
- The lower row selects COM, NAV, CRS, and XPDR input context.
- The encoder set provides range, pan, FMS outer/inner, and combined GCU478 rotary input.

## Buttons

| Control | Function |
| --- | --- |
| `DIRECT` | Direct-to |
| `MENU` | Menu |
| `FPL` | Flight plan |
| `PROC` | Procedure |
| `CLR` | Clear |
| `ENT` | Enter |
| `HOME` | Home |
| `<->` | Frequency flip / transfer-style action |
| `COM` | Select COM input context and show standby COM |
| `NAV` | Select NAV input context and show standby NAV |
| `CRS` | CRS context and show OBS values |
| `XPDR` | XPDR context and show transponder code |

## Encoders

| Encoder | Action |
| --- | --- |
| `e0` | Range and pan push |
| `e1` | Pan left/right |
| `e2` | Pan down/up |
| `e3` | FMS outer with cursor push |
| `e4` | FMS inner with cursor push |
| `e5` | GCU478 outer/inner combo, `1/2 CTR` on long press |

## Side Screens

- Left screen: range and pan hints.
- Right screen: FMS outer, FMS inner, and GCU478 combo hints.
