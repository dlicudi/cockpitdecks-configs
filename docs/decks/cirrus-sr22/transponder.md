# SR22 Transponder

## Overview { #sr22-transponder-preview }

The transponder page gives the SR22 a dedicated full-page transponder workflow instead of squeezing it into the radio page.

- The top row edits the four squawk digits directly.
- The center buttons give dedicated access to `ON`, `OFF`, `STBY`, `ALT`, `IDENT`, and `TEST`.
- `LIGHT` and `MODE` provide reply-state and current-mode feedback without sharing space with radio controls.

## What Is On The Page

| Control | Function |
| --- | --- |
| `SQUAWK0-3` | Show and increment the four code digits |
| `ON` | Transponder ON |
| `OFF` | Transponder OFF |
| `STBY` | Standby mode |
| `ALT` | ALT reporting mode |
| `IDENT` | Trigger ident |
| `TEST` | Test mode |
| `LIGHT` | Beacon-style reply/ident light |
| `MODE` | Read-only text display of the current mode |

## Highlights

- All annunciators now use sharp `v` style with dim off-state.
- The bottom-right `MODE` block mirrors the selected transponder mode in segmented text.
- This is the cleanest page for full transponder operation when you do not want radio controls mixed in.
