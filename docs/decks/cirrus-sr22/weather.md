# SR22 Weather

## Overview { #sr22-weather-preview }

Weather page for regional and atmospheric data, intended as an in-flight utility page rather than a primary navigation page.

- The page shows regional wind, visibility, dewpoint, temperature, tropopause, and QNH values.
- Pressure is available in both inHg and hPa.
- This is a read-only situational-awareness page rather than a control page.

## What Is On The Page

| Control | Function |
| --- | --- |
| `WIND DIR` | Regional wind direction |
| `WIND SPD` | Regional wind speed |
| `TEMP` | Temperature aloft |
| `VISIBILITY` | Reported visibility |
| `DEWPOINT` | Regional dewpoint |
| `TROPO ALT` | Tropopause altitude |
| `QNH (inHg)` | Pressure in inHg |
| `QNH (hPa)` | Pressure in hPa |
| `TROPO TEMP` | Tropopause temperature |

## Highlights

- This is a read-only utility page built from data widgets rather than annunciator blocks.
- Pressure is shown in both inHg and hPa.
- The page is most useful for cruise planning and quick situational awareness, not continuous scan during hand-flying.
