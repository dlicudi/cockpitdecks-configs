# SR22 Views

## Overview { #sr22-views-preview }

Quick-look view page with memorisable view slots for common cockpit viewpoints.

- The top two rows are eight quick-look view slots.
- Long-press stores the current cockpit viewpoint into the chosen slot.
- The bottom row switches between cockpit, chase, forward, and physics-model views.

## What Is On The Page

| Control | Function |
| --- | --- |
| `QUICK LOOK 0-7` | Recall quick-look views |
| long press `0-7` | Memorise the corresponding quick-look slot |
| `3D COCKPIT` | Return to cockpit view |
| `CHASE` | Chase view |
| `FORWARD` | Forward view |
| `PHYSICS` | Show physics model |

## Highlights

- The top two rows are entirely dedicated to memorisable quick-look slots.
- The bottom row switches between broader view modes rather than cockpit memorisation.
