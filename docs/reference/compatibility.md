# Compatibility

## Requirements

| Component | Version | Link |
|-----------|---------|------|
| X-Plane | 12 | [x-plane.com](https://www.x-plane.com) |
| Python | 3.10+ | [python.org](https://www.python.org) |
| XPPython3 | Latest | [Documentation](https://xppython3.readthedocs.io/en/latest/index.html) |
| Cockpitdecks | `dlicudi` fork | [GitHub](https://github.com/dlicudi/cockpitdecks) |
| Cockpitdecks X-Plane plugin | `PI_cockpitdecks.py` | Included with Cockpitdecks |
| Deck hardware | Loupedeck Live, Stream Deck XL, or other supported deck | — |

## Recommended Setup

For the most predictable behaviour, use:

- `cockpitdecks` from [dlicudi/cockpitdecks](https://github.com/dlicudi/cockpitdecks)
- `cockpitdecks-configs` from this repository
- The relevant simulator and deck extras for your hardware

## Notes

- `cockpitdecks-configs` is configuration data, not a Python package.
- Some aircraft pages may depend on custom extensions or plugins beyond the base install.
- Older helper-plugin references are obsolete for the normal installation flow.
