# Compatibility

This repository is developed and tested primarily with the following stack:

- X-Plane 12
- Python 3.10 or newer
- XPPython3
- the `dlicudi/cockpitdecks` fork
- the active Cockpitdecks X-Plane plugin `PI_cockpitdecks.py`

## Recommended setup

For the most predictable behaviour, use:

- `cockpitdecks` from `https://github.com/dlicudi/cockpitdecks`
- `cockpitdecks-configs` from this repository
- the relevant simulator and deck extras for your hardware

## Notes

- `cockpitdecks-configs` is configuration data, not a Python package.
- Some aircraft pages may depend on custom extensions or plugins beyond the base install.
- Older helper-plugin references are obsolete for the normal installation flow.
