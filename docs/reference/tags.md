# Tags

Tags are defined in each aircraft's `_docs.yaml` metadata. Common tags used across
the documentation:

| Tag | Description |
|-----|-------------|
| GA | General aviation aircraft |
| Airliner | Commercial airliner aircraft |
| Loupedeck Live | Configured for Loupedeck Live hardware |
| Stream Deck XL | Configured for Stream Deck XL hardware |
| G1000 | Aircraft with G1000 glass cockpit |
| ToLiss | ToLiss aircraft add-on |
| X-Plane 12 | Requires X-Plane 12 |
| Work in Progress | Configuration still under development |
