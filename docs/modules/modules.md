# Modules

Modules are reusable page configurations that appear across multiple aircraft. Rather than duplicating
button layouts, these common patterns are shared and adapted per aircraft.

## Common Modules

| Module | Description | Aircraft |
|--------|-------------|----------|
| **Audio Panel** | COM/NAV audio source selection and volume | Robin DR401, Cessna 172 SP, Lancair Evolution, Embraer E-Jets, Van's RV-10 |
| **Engine** | Engine instruments (EIS) adapted per engine type | All aircraft |
| **FCU** | Autopilot flight control unit — heading, altitude, VS, speed | All aircraft |
| **G1000** | Garmin G1000 soft-key and knob control | Robin DR401, Cessna 172 SP, Lancair Evolution, Embraer E-Jets, A320neo, Van's RV-10 |
| **GCU478** | Garmin GCU 478 keypad for GTN-series navigators | Cirrus SR22 |
| **PFI** | Primary flight instruments display | All aircraft |
| **Radio** | COM/NAV frequency tuning and swap | All aircraft |
| **Transponder** | Squawk code entry, mode selection, ident | All aircraft |
| **Views** | X-Plane camera view presets | Most aircraft |
| **Weather** | METAR display, wind, and atmospheric data | All aircraft |

## Engine Types

Engine data is based on the engine type number used by the G1000.[^1]

| Number | Engine Type | Example Aircraft |
|--------|-------------|-----------------|
| 1 | Single-engine fixed-pitch propeller | Cessna 172 SP |
| 2 | Single-engine constant-speed propeller | Cessna 182 |
| 3 | Single-engine turbocharged / turbo-normalized | Cessna 400 |
| 4 | Single-engine turboprop | Lancair Evolution |
| 5 | Twin-engine piston | Beechcraft Baron 58 |
| 6 | Single-engine jet | Fictional |

[^1]: [G1000 Manual (PDF)](https://x-plane.com/manuals/G1000_Manual.pdf) — MFD Engine Indication System (EIS), page 54.

## Airliner-Specific Pages

The ToLiss Airbus aircraft use a different cockpit philosophy and have specialised pages
not found in GA aircraft:

| Page | Description | Aircraft |
|------|-------------|----------|
| **ECAM** | Electronic Centralized Aircraft Monitor | A321neo, A330neo |
| **EFIS** | Electronic Flight Instrument System controls | A321neo, A330neo |
| **ADIRS** | Air Data Inertial Reference System | A321neo, A330neo |
| **Overhead panels** | Systems grouped by ATA chapter (fuel, hydraulic, electric, pneumatic, etc.) | Embraer E-Jets, A320neo, A321neo, A330neo |
