# Aircraft READMEs

Aircraft-specific documentation should live with the aircraft config in that aircraft's `README.md`.

Use those files as the primary source of truth for:

- install notes
- supported layouts
- page summaries
- aircraft-specific dependencies
- maturity or status notes
